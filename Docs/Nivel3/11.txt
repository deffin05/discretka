G=(V,E)
(*) ∀ НЕ суміжних v,u є V δ(v) + δ(u) ≥ n = |V| => G - гамільтоновий

Доведення: Припустимо, що для G виконується (*), але G - НЕ гамільтоновий. Почнемо додавати до G по одному ребру так, щоб він лишався НЕ гамільтоновим доти, доки це можливо. Отримаємо граф G': він не гамільтоновий (**), а додавання будь-якого ребра зробить його гамільтоновим. (***)
Властивість (*) зберігається у G': НЕсуміжні вершини або стали суміжними, або їхній степіінь НЕ зменшився.
Розглянемо НЕсуміжні v1, vn. Після додавання ребра (v1, vn) згідно (***) G' стане гамільтоновим => у ньому буде цикл v1, v2, ..., v(n-1), vn, v1. Вилучимо назад (v1, vn). Залишиться ланцюг v1, v2, ..., v(n-1), vn.
Для v1, vn справджується (*): δ(v1) + δ(vn) ≥ n
Розглянемо пари ребер ((v1, vn),(v2, v1)), ((v1, vn),(v2, v1)), ..., ((v2, vn),(v3, v1)), ..., ((vi, vn),(vi+1, v1)), ..., ((v(n-1), vn),(vn, v1)).
У кожній парі є максимум 1 ребро, бо якщо є обидва (vi, vn) та (v(i+1), v1), то в G' є гамільтоновий цикл v1 -> v2 -> ... -> vi -> vn -> v(n-1) -> ... -> v(i+1) -> v1, що суперечить (**)
Кожна пара додає щонайбільше 1 до δ(v1) + δ(vn). Пар всього n-1, отже δ(v1) + δ(vn) ≤ n-1 => Суперечність.
